# fix-picture-date
Powershell script to edit a datetime metadata of a picture file according to another metadata or the filename.

Datetime metadatas :
- creation date
- modification date
- taken date

filename format expected :
- "IMG_CCYYMMDD_HHIISS"
- "IMG-CCYYMMDD-WAXXXX" (whatsapp image, can't get time its name)
- "VID_CCYYMMDD_HHIISS"
